## Overview

This project is completely made using vanilla HTML and CSS. In the website are links that can be clicked in order to view different pages of the website. Within the code of this project are 5 different HTML files and utilizes only one singular CSS file. The HTML and CSS code was completely verified by using Markup Validation Service. Included in the website are 5 different pages, a home screen, hobbie page, tennis page, favorite bands page, and finally a reference page.
